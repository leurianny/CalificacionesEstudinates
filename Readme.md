# CalificacionesEstudiantes

Proyecto académico desarrollado en C# utilizando Windows Forms.
El sistema permite el cálculo y visualización de calificaciones de estudiantes
de manera sencilla y estructurada.

## Tecnologías utilizadas
- Lenguaje: C#
- Framework: .NET
- Interfaz: Windows Forms
- Entorno de desarrollo: Visual Studio

## Funcionalidades
- Ingreso de calificaciones
- Cálculo de promedio final
- Visualización de resultados
- Interfaz gráfica sencilla
